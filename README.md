## 📌 Overview

Ad click fraud is a major challenge in digital advertising, causing significant financial losses.
This project leverages **machine learning algorithms** to identify fraudulent click patterns and improve ad system reliability.

---

## 🎯 Objectives

* Detect fraudulent ad clicks accurately
* Reduce false positives and false negatives
* Analyze user click behavior
* Build a scalable fraud detection model

---

## 🧠 Machine Learning Models

* Logistic Regression
* Random Forest
* Gradient Boosting

---

## 🛠️ Tech Stack

* **Programming:** Python
* **Framework:** Django
* **Libraries:** Pandas, NumPy, Scikit-learn
* **IDE:** PyCharm
* **Dataset:** CSV-based ad click data
